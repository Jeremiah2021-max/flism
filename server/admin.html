🖥️ Run Locally on Your PC

Follow these steps to run Flism on your own machine with the same live database.

1. Requirements
Node.js v18+    →  nodejs.org
Git             →  git-scm.com
PostgreSQL 14+  →  postgresql.org  (or use Replit DB)
2. Get the code
# Download ZIP from Replit → top-right menu → Download as ZIP
# OR push to GitHub and clone:
git clone https://github.com/YOUR_USERNAME/flism.git
cd flism
3. Create server/.env
# Option A — use Replit's live database (get URL from Replit Secrets):
DATABASE_URL=postgresql://user:pass@host/db

# Option B — local PostgreSQL:
DATABASE_URL=postgresql://postgres:yourpassword@localhost:5432/flism
JWT_SECRET=your-secret-key-here
PORT=5000
4. Install & run
cd server && npm install && cd ..
cd mobile && npm install && cd ..

# Terminal 1 — backend:
cd server && node index.js

# Terminal 2 — mobile frontend:
cd mobile && npx expo start --web --port 3000

# Admin dashboard:  http://localhost:5000/admin
# Mobile app:       http://localhost:5000
💡 Tip: Copy the DATABASE_URL from Replit's Secrets tab to connect to the same live data while testing locally.
